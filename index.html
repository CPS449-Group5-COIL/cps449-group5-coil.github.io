Project Overview

This project is a cloud-based intelligent system that allows users to send and receive money securely using microservices deployed on Azure. The system supports authentication, transaction handling, and real-time balance updates.

Aegis Pay — Mobile Banking Feature

Aegis Pay is a full-stack mobile banking application built with React Native, Express, and MySQL. It supports user authentication, balance inquiries, money transfers, and real-time transaction history. The backend is deployed on Azure App Service and powered by a containerized Node.js microservice.
